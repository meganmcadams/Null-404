Heap Sort


Accessibility Settings




Description

Bubble sort is a sorting algorithm that iterates through a given array and swaps two elements if the left one is bigger than the right one. Its time complexity is O(n^2) since, in the worst case scenario, it would iterate through the array n times while swapping elements at every iteration.


Step-By-Step

1. Starting at index 0...
2. If the current element is greater than the next element, swap the elements
3. Increment index
4. Repeat steps 2 through 3 until reaching the last element
5. Repeat steps 1 through 4 until no swaps occur

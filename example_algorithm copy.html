Heapsort


Runtime Complexity

O(n log(n))

Description

Comparison-based sorting techniques based off of Binary Heap data structures. Similarities coincide with selection sort where the first minimum elements will be placed at the beginning. Unstable sorting algorithm, due to the operations being able to change the order of equivalent keys in an array. This array based heap can be used to help reduce storage and memory by overwriting inputs using no extra data structures at runtime.

Advantages - Efficiency increases logarithmically while other algorithms grow slower as # of items to sort increases in complexity. Simpler to implement, and no additional memory usage is needed to input information.

“Heapify” - is the process of creating heap data structures from binary trees using arrays to create min-heap and max-heap array based structures.
